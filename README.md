## 타입스크립트 ToDo리스트


### 프로젝트 설치

### clone repository

```
https://github.com/peeChulchul/S_TsToDo.git
```

### Install npm dependencies

```
yarn
yarn install
```

### Start dev-server

```
yarn start
```
